# Mais Acesso, Mais Saúde

## 📖 Sobre o projeto

O **Mais Acesso, Mais Saúde** é um sistema desenvolvido com o objetivo de facilitar o agendamento de consultas na rede pública de saúde. O projeto busca reduzir as dificuldades enfrentadas por pessoas que precisam esperar por longos períodos para conseguir atendimento presencial.

Além disso, a plataforma pretende beneficiar principalmente pessoas de baixa renda, moradores da zona rural e cidadãos que não têm condições de pagar consultas ou exames particulares.

## ✨ Funcionalidades

* Login de usuários;
* Agendamento de consultas;
* Consulta das datas das consultas agendadas;
* Visualização das solicitações realizadas pelo usuário.

## 🛠️ Tecnologias utilizadas

* JavaScript

## 🎯 Objetivo

Desenvolver uma solução simples e acessível para otimizar o processo de agendamento de consultas, proporcionando mais praticidade aos usuários e contribuindo para diminuir filas e deslocamentos desnecessários às unidades de saúde.
